# Traveler Profile

长期可复用旅行偏好。只记录下次旅行仍然有用的信息，不保存证件、订单号、完整聊天记录或其他敏感材料。

## Pace

- 偏轻松，不喜欢特种兵式行程。
- 情侣游优先舒服、顺路、留白。
- 每天尽量一个主区域，不为多塞景点牺牲体验。

## Food

- 用户不太能吃辣，优先不辣/少辣。
- 女朋友能吃辣，可保留少量可选辣味，但主推荐先照顾不辣。
- 不喜欢为网红店长时间排队。
- 更喜欢街头巷尾但干净靠谱的店，不要全是热门打卡店。

## Hotel

- 不追求豪华，优先干净卫生、好评稳定、位置方便。
- 酒店应尽量贴近当天主要活动区域，减少路途时间。
- 订房优先级：近公共交通 > 近主要活动区 > 近半年卫生/隔音评价稳定 > 可取消。

## Transport

- 城市旅行优先减少折返和长距离通勤。
- 能把交通变成体验时可以保留，例如轮渡、观景路线。

## Output Preference

- 需要真实地图行程图，优先交付可手机打开的 HTML 地图页。
- 适合把每次旅行单独归档，长期偏好和单次需求分开保存。
