# 2606 Changsha Itinerary

## Base Area

优先订桔子酒店，搜索范围放在五一广场 / 黄兴广场 / IFS / 芙蓉广场一带。理由：下雨天中心酒店价值更高，能减少打车和湿鞋；五一广场和芙蓉广场都在地铁 2 号线上，去长沙南站顺，去核心商圈也方便。

不建议为了高铁方便住长沙南站附近，因为这几天主要活动都在市中心和岳麓/洋湖方向，住南站会让每天通勤变长。

## Day 1 - 6月19日：高铁抵达与五一广场轻松夜逛

- 13:27-16:19：深圳高铁到长沙南站。
- 16:30-17:20：长沙南站坐地铁 2 号线到五一广场 / 芙蓉广场酒店区域，或雨大时打车。
- 17:30-18:20：酒店办理入住、放行李。
- 18:40：酒店附近吃晚餐，优先不辣/少辣米粉、小吃或家常菜。
- 19:30：路过短队的茶颜悦色买一杯。
- 20:00：黄兴路 / 太平老街短距离散步；雨大就只逛商场和酒店附近。

策略：第一天只进入长沙状态，不安排硬核景点。

## Day 2 - 6月20日：湖南博物院与雨天市中心

- 10:10：从酒店出发。
- 10:45-13:30：湖南博物院，重点看马王堆汉墓相关展陈。
- 13:40：省博附近或回五一广场午餐，不为单店排队。
- 15:00：回酒店休息，或去 IFS / 国金中心避雨、补茶颜悦色。
- 18:30：五一广场 / 黄兴广场晚餐，找少辣湘菜、粉面或商场餐。
- 20:00：如果雨小，短去杜甫江阁或湘江边看夜景；雨大就取消。

策略：这天预报雨量较大，湖南博物院是主轴，户外只作为短窗口。

## Day 3 - 6月21日：雨天版文化与影像馆

- 10:10：从酒店出发。
- 10:50-12:20：若雨小，去岳麓书院 / 爱晚亭短线；若雨大，直接改去室内。
- 12:30：湖南大学 / 岳麓周边或市中心午餐，依排队和雨势决定。
- 14:30-17:00：雨天主推谢子龙影像艺术馆 / 李自健美术馆方向，适合拍照和室内停留。
- 18:30：回五一广场晚餐。
- 20:00：雨小才考虑橘子洲近段；不去青年雕塑远端。

策略：这天预报降水很强，不把橘子洲当硬任务。能玩到岳麓书院最好，雨大就用影像馆/美术馆保体验。

## Day 4 - 6月22日：酒店附近吃饭，去长沙南站

- 10:00：起床整理、退房或寄存行李。
- 10:30：酒店附近吃粉面/早午餐，选干净、不排队、能做不辣的店。
- 11:30：回酒店拿行李。
- 12:00-12:50：前往长沙南站；雨大时建议直接打车，或地铁 2 号线。
- 14:19-17:08：高铁回深圳。

策略：最后一天不安排参观景点，只保证吃得舒服、返程稳。

## Food Notes

- 点单关键词：不辣、少辣、辣椒另放。
- 粉面优先原汤粉、肉丝粉、牛肉粉。
- 湘菜先点汤、蒸蛋、青菜、非重辣肉菜。
- 排队超过 15-20 分钟直接换店。
- 茶颜悦色不用死守某一家，看到短队再买。

- 6/20：25.2-28.3°C，降水概率 96%，约 43.8mm。
- 6/21：24.5-26.8°C，降水概率 90%，约 80.4mm。
- 6/22：24.2-25.4°C，降水概率 84%，约 79.2mm。
