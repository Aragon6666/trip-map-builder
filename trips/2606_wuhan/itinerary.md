# 2606 Wuhan Itinerary

## Base Area

建议住在江汉路 / 循礼门 / 江汉关一带。理由：第一晚可以步行看汉口江滩，最后一天去机场方便，去黄鹤楼、省博、东湖时跨江一次即可，不用每天换酒店。

## Day 1 - 6月19日：抵达与汉口江滩

- 下午：深圳飞武汉，抵达武汉天河国际机场。
- 傍晚：前往江汉路 / 循礼门酒店中心，办理入住。
- 晚上：江汉路步行街轻松散步。
- 晚餐/小吃：三镇民生甜食馆或附近不排队小吃。
- 夜景：江汉关大楼、汉口江滩。

策略：第一天只进入状态，不安排硬核景点。若航班延误，保留江滩夜景即可。

## Day 2 - 6月20日：黄鹤楼、粮道街、昙华林、轮渡

- 上午：黄鹤楼公园。
- 中午：司门口 / 粮道街午餐区，优先豆皮、烧麦、汤粉、藕汤等不太辣选择。
- 下午：昙华林慢走拍照。
- 傍晚：中华路码头坐轮渡到武汉关码头。
- 晚餐：回汉口老街区找湖北菜/家常菜，不为网红店排长队。

策略：武昌一日集中处理，不把黄鹤楼和东湖硬塞在同一天。

## Day 3 - 6月21日：湖北省博物馆与东湖

- 上午：湖北省博物馆，重点看曾侯乙编钟、越王勾践剑、楚文化展。
- 可选：湖北美术馆，作为高温/下雨备选。
- 下午：东湖听涛 / 梨园入口轻松散步。
- 茶歇：东湖附近找有座位、不排队的咖啡或饮品。
- 晚餐：楚河汉街或回江汉路，选择不辣/少辣湖北菜或商场简餐。

策略：省博是室内核心点，东湖只保留轻松湖边体验，不深度骑行。

## Day 4 - 6月22日：汉口轻散步与返程

- 上午：古德寺拍照，或改为更近的黎黄陂路街头博物馆。
- 中午：江汉路附近午餐或打包。
- 下午：前往武汉天河国际机场返程。

策略：最后一天不跨远区，不去东湖/远郊，给返程留足余量。
